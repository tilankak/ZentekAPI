🛍️ Zentek Web API (.NET 6+)

This project is a RESTful Web API built using .NET 6+ that provides basic product management functionality, including secure endpoints and filtering capabilities.

📌 Features
✅ Health check endpoint (public/anonymous)
🔐 Secured product management endpoints
📦 Create new products
📄 Retrieve all products (JSON format)
🎨 Filter products by colour
🧪 Unit and integration tests included
